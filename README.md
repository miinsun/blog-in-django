# blog-in-django
python django를 활용한 웹 개발 Blog 만들기            
[바로가기 링크](http://minsun.pythonanywhere.com/) 

## 개발환경
- server    
    - pythonanwhere 호스팅
- db    
    - sqlite
- python    
    - python 3.8.5      
    - Django2.0.13     
    - djangorestframework     
- css   
    - bootstrap 3.2.0
    
## 사용 가이드
회원 가입 버튼을 눌러 회원 가입 후 게시물 작성, 수정 등 블로그 기능을 사용 할 수 있습니다.      
댓글 작성은 한 게시물 당 3개씩 작성 할 수 있도록 제한을 걸어두었습니다.

- djangoGirls의 공식 문서를 바탕으로 제작하였습니다.
